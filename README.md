# Angular on Coffee

FER 2013/2014, 1st year of Master degree, university course Seminar: 
"AngularJS with CoffeeScript"

DEMO with Sails.js backend.


## How to start
### Requirements
> sudo apt-get install nodejs
> sudo apt-get install npm
> npm install grunt -g
> npm install bower -g

### Install
> cd frontend
> npm install
> bower update
> 
> cd ../backend
> npm install

### Start apps
1. open 2 terminals
2. in first terminal start backend app
    a) sails lift
3. in second terminal start frontend app
    a) grunt serve
